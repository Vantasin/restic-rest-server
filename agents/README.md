# Agents Directory

This directory contains agent-facing repository guidance only. Human-facing
deployment and operations documentation belongs in [`../Docs/`](../Docs/README.md)
and the root [`../README.md`](../README.md).

## Layout

- [`workflows/`](./workflows/README.md): repeatable agent procedures
- [`context/`](./context/README.md): repo structure and component context
- [`rules/`](./rules/README.md): non-negotiable constraints for changes

## Entry Point

Use [`../AGENTS.md`](../AGENTS.md) as the thin router into this directory.
