# Workflows Directory

This directory contains the agent workflows used to review and extend the repo
without drifting from its source-of-truth structure.

## Files

- [`review.md`](./review.md): review workflow index
- [`agent-review.md`](./agent-review.md): check `AGENTS.md` and `agents/`
- [`repo-review.md`](./repo-review.md): check repo behavior and human docs
- [`documentation.md`](./documentation.md): update docs in the right order
- [`change-logging.md`](./change-logging.md): record what changed and how it
  was verified
- [`new-area.md`](./new-area.md): add a new major repo area cleanly
