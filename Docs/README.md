# Docs Directory

This directory contains the human-readable documentation for the repository.
Use the root [`README.md`](../README.md) as the quick-start entry point, then
follow the docs here for deployment, configuration, storage, security, and
operations details.

## What Lives Here

- [`DEPLOYMENT.md`](./DEPLOYMENT.md): first deployment flow and update workflow
- [`CONFIGURATION.md`](./CONFIGURATION.md): tracked env contract and defaults
- [`STORAGE.md`](./STORAGE.md): persistent storage layout and path choice
- [`SECURITY.md`](./SECURITY.md): threat model and deployment security guidance
- [`OPERATIONS.md`](./OPERATIONS.md): day-two commands, verification, and hooks
- [`decisions/README.md`](./decisions/README.md): lightweight decision log
